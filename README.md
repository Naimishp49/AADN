
## Subject Syllabus: 2501CS634 – Advanced .NET with Modern Architectures

## Basic Information

- **Program**: Bachelor of Technology (BTech)  
- **Semester**: 6  
- **Course Type**: Professional Elective  
- **Prerequisites**:  
  - Basic understanding of C# programming  
  - ASP.NET Core fundamentals  
  - SQL database operations  
  - REST API concepts  
- **Effective From Academic Year**: 2025–26  
- **Rationale**:  
  This course enables students to acquire hands-on, industry-relevant skills by integrating real-world concepts such as background processing, event-driven communication, cloud deployment, and real-time messaging. By working with widely adopted tools and frameworks in the .NET ecosystem, students gain practical exposure to the development practices used in enterprise-grade solutions.

## Course Outcomes (COs)

At the end of this course, students will be able to:

- **CO1**: Explain microservices concepts, design patterns, and communication methods.  
- **CO2**: Implement Dapper CRUD, caching, and structured logging.  
- **CO3**: Apply JWT security, rate limiting, and write basic unit tests.  
- **CO4**: Deploy .NET apps to cloud and build real-time features using SignalR.  
- **CO5**: Use Hangfire for background jobs and RabbitMQ for event-driven messaging.
